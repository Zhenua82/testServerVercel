Фронт по работе с серверами на Node.js

Бэкэнд выполнен как на нативном Node.js, так и c использованием фреймворка Express, так и в среде в среде Vercel Serverless Functions (без использования express). Для хранения данных - использована база данных Neon (PostgreSQL), для работы с медиафайлами использован сервис: cloudinary.com

Варианты серверов:

▶ Работа с сервером render.com (c использованием express) и бд Neon
▶ Работа с сервером vercel.com (в среде Vercel Serverless Functions без использования express) и бд Neon ▶ Простой запрос на сервере vercel.com в отдельном API (в среде Vercel Serverless Functions)
▶ Сложные запросы на сервере vercel.com (без использования express, на нативном Node.js)
