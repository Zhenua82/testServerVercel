Работа с серверами на Node.js

В этом проекте представлены различные варианты реализации backend-части: нативный Node.js, Express и Serverless-функции на Vercel.

Используется PostgreSQL (Neon) и Cloudinary для хранения изображений.

Демонстрация:

▶ Сервер на Render (Express + PostgreSQL) ▶ Сервер на Vercel (Serverless + PostgreSQL) ▶ Простой API-запрос (Serverless, без хранения данных) ▶ Сложные API-запросы (Node.js, сервер на Vercel, хранения данных в оперативной памяти)
