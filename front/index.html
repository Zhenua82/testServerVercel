Работа с серверами на Node.js

В этом проекте представлены различные варианты реализации backend-части: нативный Node.js, Express, а также Serverless-функции на Vercel.

В качестве базы данных используется PostgreSQL (Neon), а для хранения изображений — Cloudinary.

Демонстрация работы:

▶ Сервер на Render (Express + PostgreSQL) ▶ Сервер на Vercel (Serverless Functions + PostgreSQL) ▶ Простой API-запрос (Vercel) ▶ Сложные API-запросы (чистый Node.js)
